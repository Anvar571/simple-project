## Installation

```bash
$ pnpm install
```

## Running the app with docker

```bash
# development
$ pnpm run start:docker:dev

# watch mode
$ pnpm run start:docker:prod

```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

```
